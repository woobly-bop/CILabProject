# User Manual

This manual explains how to use the Jenkins jobs in this project.

## Using Freestyle Job
1. Open Jenkins Dashboard
2. Click on Freestyle-CI-Job
3. Click Build Now
4. View Console Output for build logs
5. Check Test Results for JUnit report

## Using Multibranch Pipeline
1. Open Multibranch-CI-Pipeline
2. Click Scan Multibranch Pipeline
3. Select a branch to view pipeline execution

## Viewing Artifacts
1. Open a successful build
2. Click Artifacts
3. Download the generated JAR file

## Monitoring Builds
- Check build status (Success/Failure)
- Review logs in Console Output
