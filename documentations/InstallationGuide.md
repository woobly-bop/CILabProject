# Installation Guide

This guide explains how to install and configure Jenkins for this project.

## Prerequisites
- Windows OS
- Java JDK 17
- Apache Maven
- Git
- Python (optional)

## Step 1: Install Java
Download and install Java JDK 17 and configure JAVA_HOME.

## Step 2: Install Maven
Download and extract Apache Maven and configure MAVEN_HOME.

## Step 3: Install Jenkins
Download Jenkins LTS version and complete the setup wizard.

## Step 4: Install Required Plugins
Install the following plugins:
- Pipeline
- Git
- Maven Integration
- JUnit
- Docker
- Slack Notification

## Step 5: Configure Jenkins Tools
Configure:
- JDK
- Maven
- Git

## Step 6: Connect Jenkins to GitHub
Create Jenkins jobs and provide the GitHub repository URL.

Installation is now complete.
