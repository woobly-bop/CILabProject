# Troubleshooting Guide

## Issue: mvn command not found
Solution: Install Maven and configure MAVEN_HOME.

## Issue: JAVA_HOME not defined
Solution: Set JAVA_HOME in Jenkins and system environment variables.

## Issue: GitHub repository not cloning
Solution: Check repository URL and credentials.

## Issue: Tests not running
Solution: Ensure test files exist under src/test/java.

## Issue: Python script not found
Solution: Ensure build.py exists in scripts folder and Python is installed.

## Issue: Jenkins pipeline fails
Solution: Check Jenkinsfile syntax and environment configuration.
